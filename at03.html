Seção 2 — Ranking de tecnologias

  1. HTML

    É a base da construção das páginas web.

  2. CSS

    Responsável pela aparência e estilo das páginas.

  3. JavaScript

    Permite adicionar interatividade às páginas.

  4. HTTP

    Protocolo utilizado para comunicação entre cliente e servidor.

  5. API

    Permite a comunicação entre diferentes sistemas.

Seção 3 — Glossário de termos técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem utilizada para estilizar páginas web.
JavaScript
Linguagem de programação para páginas web.
Permite adicionar interatividade.
DOM
Representação da estrutura de um documento HTML.
API
Interface de programação de aplicações.
Permite integração entre sistemas.
HTTP
Hypertext Transfer Protocol
Protocolo utilizado na comunicação entre navegador e servidor.
URL
Uniform Resource Locator
Endereço utilizado para localizar recursos na internet.
DNS
Sistema responsável por converter nomes de domínio em endereços IP.
